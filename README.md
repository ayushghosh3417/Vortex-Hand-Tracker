# 🚀 VORTEX AR — Hand Tracking Interactive Experience

An interactive **web-based AR experience** where users can control visuals using real-time hand gestures 👋

Built using **JavaScript + MediaPipe + Canvas API**, this project blends computer vision with creative frontend animations.

---


---

## ✨ Features

* 👋 Real-time **Hand Tracking** using MediaPipe
* ✌️ Gesture Detection (Pinch, Open Hand, Fist)
* ⚡ Dynamic Particle Effects & Shockwaves
* 🌈 Multiple Visual Themes (Cyberpunk, Galaxy, Lava, Ocean)
* 🎨 Glassmorphism UI Design
* 🌌 Animated Landing Page (Moving Stars Background)
* 🔊 Interactive Audio Feedback

---

## 🛠️ Tech Stack

* **JavaScript (Core Logic)**
* **HTML5 Canvas (Rendering & Effects)**
* **CSS (Glass UI + Styling)**
* **MediaPipe Hands (Computer Vision)**

## 🎯 How It Works

* Uses **MediaPipe Hands** to detect hand landmarks
* Tracks finger positions in real-time
* Applies physics-based animations on Canvas
* Gesture detection triggers:

  * ⚡ Shockwaves (Pinch)
  * ✨ Particle effects (Fingertips)
  * 🌐 Dynamic connections between hands
